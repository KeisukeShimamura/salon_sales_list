# サロン営業リスト管理ソフト
## やりたいこと
* いろんなサイトからサロンの情報や、掲載している求人情報などをスクレイピングする
  * HOT PAPPER BEAUTY
  * リクエストQJ
  * リジョブ
  * etc..
* スクレイピングした情報を突合して１つにまとめる
  * どこに掲載されているかなども持たせる
* 機械学習でデータ分析する
  * 教師なし学習。営業成約率の高いサロンを分類する
    * いろんな媒体を使っているか？
    * 店舗拡大しようとしているか？
    * 上記などを判定して採用に困っているサロンを判定

## 開発
### スクレイピン
* 必要パッケージ
  * scrapy
  * beautifulsoup4
* 実行コマンド

```
scrapy crawl hot_papper_beauty -o data.csv
```
